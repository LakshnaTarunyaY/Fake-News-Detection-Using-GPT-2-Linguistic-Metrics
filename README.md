# Fake-News-Detection-Using-GPT-2-Linguistic-Metrics
AI-powered Fake News Detection using GPT-2, Hugging Face Transformers, PyTorch, and Scikit-Learn. Extracted LLM-based linguistic features, trained a Logistic Regression classifier, and developed a Streamlit application for news classification and visualization.

Key Features
Extracted GPT-2 based perplexity scores to measure text predictability and linguistic complexity.
Computed token prediction accuracy using language model outputs.
Generated numerical feature representations from news articles.
Trained a Logistic Regression classifier on extracted LLM-based features.
Built an interactive Streamlit application for dataset upload, analysis, and visualization.
Visualized perplexity distributions, token accuracy distributions, and feature relationships.
