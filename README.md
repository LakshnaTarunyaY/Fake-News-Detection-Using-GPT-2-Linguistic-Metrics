# Fake-News-Detection-Using-GPT-2-Linguistic-Metrics
AI-powered Fake News Detection using GPT-2, Hugging Face Transformers, PyTorch, and Scikit-Learn. Extracted LLM-based linguistic features, trained a Logistic Regression classifier, and developed a Streamlit application for news classification and visualization.

Key Features:

1.Extracted GPT-2 based perplexity scores to measure text predictability and linguistic complexity.

2.Computed token prediction accuracy using language model outputs.

3.Generated numerical feature representations from news articles.

4.Trained a Logistic Regression classifier on extracted LLM-based features.

5.Built an interactive Streamlit application for dataset upload, analysis, and visualization.

6.Visualized perplexity distributions, token accuracy distributions, and feature relationships.

Technologies Used:

Python,
GPT-2,
Hugging Face Transformers,
PyTorch,
Scikit-Learn,
Logistic Regression,
Streamlit,
Pandas,
Matplotlib

Workflow:

1.Upload news dataset.

2.Extract perplexity and token prediction accuracy using GPT-2.

3.Generate feature vectors from linguistic metrics.

4.Train and evaluate a Logistic Regression classifier.

5.Visualize feature distributions and classification performance.
